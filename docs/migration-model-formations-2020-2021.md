# Migration de modèle mna Formation 2020->2021

## Légende 

### Modifiée
en orange soit le nom de la clé à changée soit le type de valeur à changé
```diff
{
! oldKey: value,
+ newKey: value,
}
```

### Ajoutée
en vert les nouvelles clés
```diff
{
+ newKey: value,
}
```

### Supprimée
en rouge les clés définitivement supprimées
```diff
{
- oldKey: value,
}
```
